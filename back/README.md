To install dependencies:
```sh
bun install
```

To run:
```sh
bun run dev
```

## DB seed in dev mode

Run the local version of the DB
```sh
bun db:local
```

Run the push command to make the migration in dev mode
```sh
bun db:push
```

Run the seed command to insert dummy data
```sh
bun db:seed
```

open http://localhost:3000
